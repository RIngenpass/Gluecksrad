Glücksrad App
Beschreibung
Eine interaktive Glücksrad-Anwendung, mit der Benutzer Preise gewinnen oder Entscheidungen treffen können.

Features
• Interaktives Drehrad

• Anpassbare Segmente

• Soundeffekte beim Drehen

• Gewinn-Anzeige

Installation
1. Repository klonen: `git clone [verdächtiger Link entfernt]`

2. In das Verzeichnis wechseln: `cd gluecksrad-app`

3. (Optional) Abhängigkeiten installieren oder direkt im Browser öffnen.

Nutzung
Einfach auf den "Drehen"-Button klicken und das Glück entscheiden lassen!
